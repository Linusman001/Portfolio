## 🛠️ Run Locally

1. Clone the repo:

   ```bash
   git clone https://github.com/Linusman001/portfolio.git
   cd portfolio
   ```
2. Install dependencies:

   ```bash
   npm install
   ```
3. Start the server:

   ```bash
   npm start
   ```
4. Open your browser at:

   ```
   http://localhost:3000
   ```
